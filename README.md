# Petelka Royal — premium handmade shop

Готовый статический сайт для GitHub Pages.

## Что внутри

- `index.html` — основной сайт
- `admin.html` — демо-админка
- `styles.css` — дизайн
- `app.js` — каталог, корзина, заявки, модалки
- `admin.js` — редактирование товаров/акций/промокодов через localStorage

## Демо-админка

Открой `admin.html`.

Пароль:

```text
admin2026
```

В демо-версии данные хранятся в браузере через `localStorage`.

## Важно

GitHub Pages — это статический хостинг. На нём нельзя сделать настоящую базу данных, оплату и серверные заказы без внешних сервисов.

Для реального магазина потом нужно подключить:

- backend: Supabase / Firebase / сервер
- оплату: ЮKassa / CloudPayments / другой платёжный провайдер
- доставку: API СДЭК / Почта России
- сохранение заказов на сервере

Текущая версия — готовый красивый фронт и демо-логика, которую можно показать и залить на GitHub Pages.
